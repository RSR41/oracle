# Screenshot Guide for Flutter Porting

로컬에서 React 앱 실행 후(`npm run dev`), 크롬에서 확인해야 할 핵심 화면 리스트입니다.
예상 URL: `http://localhost:5173 /` (Vite 기본)

## 1. Onboarding (최초 실행)
- [ ] **첫 화면**: 앱 실행 시 가장 먼저 뜨는 소개 화면.
- [ ] **단계별 진행**: [다음] 버튼을 눌러가며 각 단계 캡처.

## 2. Main Tabs (하단 탭)
- [ ] **홈 (Home)**
    - 상단 날짜/인사말 헤더
    - "오늘의 운세" 카드 (점수 표시)
    - 바로가기 그리드 (만세력, 궁합 등)
- [ ] **운세 (Fortune)**
    - 상단 탭 (전체/오늘/만세력 등)
    - 리스트형 카드 배치 확인
- [ ] **궁합 (Compatibility)**
    - 파트너 추가/선택 UI
    - "궁합 보기" CTA 버튼
- [ ] **기록 (History)**
    - 타임라인 형태의 기록 리스트
- [ ] **내 정보 (Profile)**
    - 프로필 이미지, 닉네임, 설정 메뉴 리스트

## 3. Detail Flows (상세 진입)
- [ ] **오늘의 운세 상세**
    - 홈 > "오늘의 운세" 카드 클릭
    - 결과 화면의 텍스트 배치 및 하단 버튼
- [ ] **만세력 (Calendar)**
    - 운세 탭 > "만세력" 카드 클릭
    - 달력 UI 및 날짜 선택 효과
- [ ] **궁합 결과**
    - 궁합 탭 > "궁합 확인하기" > 이름 입력 > 결과 화면
    - 두 사람의 점수와 분석 텍스트
- [ ] **설정 (Settings)**
    - 내 정보 탭 > 우측 상단 톱니바퀴 (또는 메뉴)
    - 테마(다크모드) 변경 토글 테스트 & 캡처

## 4. UI Components Check
- [ ] **Dark Mode**: 설정에서 다크모드 켜고 `Home` 화면 캡처 (Flutter 테마 매핑용).
- [ ] **Toaster**: 아무 버튼이나 눌렀을 때 하단에 뜨는 알림 메시지 캡처.
- **Micro-interactions**: 탭 전환 시 부드러운 전환 효과 등은 영상 기록 권장.
