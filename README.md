# Oracle

Monorepo for **Oracle**: a cross-platform fortune app (Android, iOS, PWA) with a shared backend.

## Structure
- `backend/` : NestJS + Prisma + PostgreSQL API
- `pwa/`     : Next.js PWA (NFC tag entry)
- `apps/android/` : Android app (Kotlin + Compose)
- `apps/ios/`     : iOS app (SwiftUI)
- `docs/`    : documentation
