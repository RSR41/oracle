# Smoke Test Checklist (Release Mode)

이 체크리스트는 `BETA_FEATURES=false` 설정으로 빌드된 출시 버전의 무결성을 검증하기 위한 것입니다.

> 스토어 메타데이터/정책 입력 상태는 [STORE_SUBMISSION_CHECKLIST.md](./STORE_SUBMISSION_CHECKLIST.md)에서 함께 관리하세요.

## 1. 앱 설치 및 브랜딩
- [ ] **아이콘 확인**: 홈 화면의 앱 아이콘이 사전에 설정한 Placeholder(또는 실제 아이콘)로 노출되는가?
- [ ] **앱 이름 확인**: 아이콘 아래 표시 이름이 "Oracle 사주"인가?
- [ ] **스플래시 화면**: 앱 구동 시 배경색(#F5F1E9)과 중앙 이미지가 포함된 스플래시가 노출되는가?

## 2. 베타 격리 검증 (BETA_FEATURES=false)
- [ ] **하단 탭**: 'Meeting', 'Compat' 탭이 사라지고 [Home, Fortune, History, Profile] 4개만 보이는가?
- [ ] **운세(Fortune) 화면**: 소개팅, 이상형 그리기 메뉴가 보이지 않는가?
- [ ] **MVP 5개 기능 노출**: 사주, 만세력, 타로, 꿈해몽, 관상 메뉴가 정상 표시되는가?
- [ ] **경로 진입 차단**: (딥링크 등 비정상 경로) `/meeting` 등으로 진입 시 홈으로 리다이렉트되는가?

## 3. MVP 기능 (5개 - 항상 접근 가능)
- [ ] **오늘의 운세(사주)**: 메인 홈 또는 운세 탭에서 오늘의 운세 결과가 정상 출력되는가?
- [ ] **만세력(캘린더)**: 캘린더 화면이 정상 로드되고 일별 운세 데이터가 조회되는가?
- [ ] **타로**: 타로 화면 진입 → 카드 뽑기 → 결과 출력이 정상 동작하는가?
- [ ] **꿈해몽**: 꿈 입력 → 해석 결과 출력이 정상 동작하는가?
- [ ] **관상(얼굴 분석)**: 이미지 선택/촬영 → 분석 결과 출력이 정상 동작하는가?
- [ ] **데이터 저장**: 운세 결과 조회 후 히스토리(History) 탭에 기록이 남는가?

## 4. 히스토리 상세
- [ ] **기록 리스트**: 과거 사주/운세 기록들이 리스트 형태로 노출되는가?
- [ ] **상세 진입**: 기록 클릭 시 상세 결과 화면으로 이동하며, 이전 데이터가 정확히 표시되는가?

## 5. 법적 링크 (신규)
- [ ] **설정 화면 진입**: 프로필 탭에서 설정 화면으로 이동 가능한가?
- [ ] **이용약관 항목**: "이용약관" 메뉴가 표시되는가?
- [ ] **개인정보처리방침 항목**: "개인정보처리방침" 메뉴가 표시되는가?
- [ ] **이용약관 탭**: 외부 브라우저 열림 또는 placeholder 안내 메시지 표시
- [ ] **개인정보처리방침 탭**: 외부 브라우저 열림 또는 placeholder 안내 메시지 표시
- [ ] **오픈소스 라이선스**: "오픈소스 라이선스" 탭 시 라이선스 페이지 표시

## 6. 안정성
- [ ] **내비게이션**: 탭 간 이동 시 딜레이나 크래시가 없는가?
- [ ] **뒤로가기 제어**: 상세 화면에서 뒤로가기 시 이전 리스트로 정확히 복귀하는가?
