# Smoke Test Checklist (Release Mode)

이 체크리스트는 `BETA_FEATURES=false` 설정으로 빌드된 출시 버전의 **실제 현재 동작** 기준입니다.

## 재검증 기록
- 검증 기준 브랜치: 현재 작업 브랜치
- 문서 동기화 포인트
  - 하단 탭: `[Home, Fortune, History, Profile]` (Meeting/Compat 숨김)
  - Fortune 탭 노출 기능: 오늘의 운세, 만세력, 사주(프로필 유도), 타로
  - Dream/Face/Meeting/Compat: 베타 전용(스토어 릴리즈 기본 비노출)

> 스토어 메타데이터/정책 입력 상태는 [STORE_SUBMISSION_CHECKLIST.md](./STORE_SUBMISSION_CHECKLIST.md)에서 함께 관리하세요.

## 1. 앱 설치 및 브랜딩
- [ ] **아이콘 확인**: 홈 화면 앱 아이콘이 설정한 아이콘으로 노출되는가?
- [ ] **앱 이름 확인**: 아이콘 아래 표시 이름이 "Oracle 사주"인가?
- [ ] **스플래시 화면**: 앱 구동 시 배경색(#F5F1E9) + 중앙 스플래시 이미지가 노출되는가?

## 2. 베타 격리 검증 (`BETA_FEATURES=false`)
- [ ] **하단 탭 개수**: [Home, Fortune, History, Profile] 4개만 노출되는가?
- [ ] **Meeting/Compat 숨김**: Meeting/Compat 탭이 보이지 않는가?
- [ ] **Fortune 탭 베타 메뉴 숨김**: Dream 메뉴가 보이지 않는가?
- [ ] **경로 진입 차단**: `/meeting`, `/compatibility` 직접 진입 시 `/home`으로 리다이렉트되는가?

## 3. 릴리즈 기본 기능 검증 (상시 접근)
- [ ] **오늘의 운세(사주)**: 오늘 날짜 기준 운세(점수/메시지)가 출력되는가?
- [ ] **만세력(캘린더)**: 캘린더 화면이 정상 로드되고 일별 점수/간지가 표시되는가?
- [ ] **타로**: 타로 화면 진입 → 셔플/카드 뽑기 → 결과 출력이 정상 동작하는가?
- [ ] **사주 분석 진입**: Fortune 탭의 사주 카드 선택 시 Profile 탭(사주 입력/결과)로 이동하는가?
- [ ] **데이터 저장**: 오늘의 운세/타로 결과 저장 후 History 탭에 기록이 남는가?

## 4. 히스토리 상세
- [ ] **기록 리스트**: 과거 기록이 리스트 형태로 노출되는가?
- [ ] **상세 진입**: 기록 클릭 시 상세 화면으로 이동하고 저장된 값이 정확한가?
- [ ] **필터 동작**: 타입 필터(ALL/FORTUNE/TAROT/DREAM/FACE/MEETING 등)가 정상 동작하는가?

## 5. 설정/법적 링크
- [ ] **설정 화면 진입**: 프로필 탭에서 설정 화면으로 이동 가능한가?
- [ ] **이용약관 항목**: "이용약관" 메뉴가 표시되는가?
- [ ] **개인정보처리방침 항목**: "개인정보처리방침" 메뉴가 표시되는가?
- [ ] **이용약관 탭**: 외부 브라우저 열림 또는 placeholder 안내 메시지 표시
- [ ] **개인정보처리방침 탭**: 외부 브라우저 열림 또는 placeholder 안내 메시지 표시
- [ ] **오픈소스 라이선스**: 라이선스 페이지가 표시되는가?

## 6. 안정성
- [ ] **내비게이션**: 탭 전환/화면 이동 시 딜레이나 크래시가 없는가?
- [ ] **뒤로가기 제어**: 상세 화면에서 뒤로가기 시 이전 화면으로 정확히 복귀하는가?
