# Oracle 사주 개인정보처리방침

**최종 수정일**: 2026년 2월 7일  
**시행일**: 2026년 2월 7일

---

## 1. 개인정보의 수집 및 이용 목적

**ORACLE** (이하 "회사")는 Oracle 사주 앱(이하 "앱")을 통해 다음의 목적으로 개인정보를 처리합니다:

| 목적 | 설명 |
|------|------|
| 서비스 제공 | 사주, 타로, 꿈해몽, 관상 등 운세 분석 서비스 제공 |
| 기록 관리 | 이용자가 조회한 운세 결과를 기기 내 저장하여 히스토리 기능 제공 |

---

## 2. 수집하는 개인정보 항목

### 2.1 사용자 입력 정보
| 항목 | 수집 여부 | 저장 위치 | 서버 전송 |
|------|----------|----------|----------|
| 생년월일 | ✅ 수집 | 기기 내 로컬 저장 | ❌ 없음 |
| 출생 시간 | ✅ 수집 (선택) | 기기 내 로컬 저장 | ❌ 없음 |
| 이름/닉네임 | ✅ 수집 (선택) | 기기 내 로컬 저장 | ❌ 없음 |
| 성별 | ✅ 수집 (선택) | 기기 내 로컬 저장 | ❌ 없음 |

### 2.2 관상 기능 (얼굴 이미지)
| 항목 | 수집 여부 | 저장 위치 | 서버 전송 |
|------|----------|----------|----------|
| 얼굴 이미지 | ✅ 수집 (관상 기능 이용 시) | **기기 내 로컬 저장만** | ❌ **서버 전송 없음** |

> **중요**: 관상 기능에서 사용하는 얼굴 이미지는 **외부 서버로 전송되지 않습니다**. 모든 이미지 처리는 사용자의 기기에서만 이루어집니다.

### 2.3 자동 수집 정보
| 항목 | 수집 여부 | 설명 |
|------|----------|------|
| 기기 식별자 | ❌ 수집하지 않음 | - |
| IP 주소 | ❌ 수집하지 않음 | - |
| 위치 정보 | ❌ 수집하지 않음 | - |

---

## 3. 개인정보의 보관 및 파기

### 3.1 보관 기간
- **기기 내 저장 데이터**: 사용자가 앱을 삭제하거나 데이터를 직접 삭제할 때까지 보관

### 3.2 파기 방법
- **앱 삭제**: 앱을 삭제하면 모든 로컬 데이터가 자동으로 삭제됩니다
- **데이터 초기화**: 앱 내 설정에서 데이터를 직접 삭제할 수 있습니다
- **히스토리 삭제**: 저장된 운세 기록을 개별 삭제할 수 있습니다

---

## 4. 개인정보의 제3자 제공

회사는 이용자의 개인정보를 **제3자에게 제공하지 않습니다**.

| 항목 | 내용 |
|------|------|
| 제3자 제공 | ❌ 없음 |
| 외부 서버 전송 | ❌ 없음 |
| 광고 네트워크 연동 | ❌ 없음 |
| 분석 서비스 연동 | ❌ 없음 |

---

## 5. 개인정보 처리의 위탁

회사는 개인정보 처리를 **외부 업체에 위탁하지 않습니다**.

---

## 6. 이용자의 권리

이용자는 다음의 권리를 행사할 수 있습니다:

| 권리 | 행사 방법 |
|------|----------|
| 정보 열람 | 앱 내 히스토리 화면에서 저장된 데이터 확인 |
| 정보 삭제 | 앱 내에서 개별 기록 삭제 또는 앱 삭제로 전체 데이터 삭제 |
| 정보 수정 | 앱 내 프로필/설정에서 입력 정보 수정 |

---

## 7. 개인정보의 안전성 확보 조치

회사는 다음의 조치를 통해 개인정보를 보호합니다:

1. **로컬 저장**: 모든 데이터는 사용자 기기 내에만 저장됩니다
2. **서버 미사용**: 외부 서버로 개인정보를 전송하지 않습니다
3. **암호화 저장**: SQLite 데이터베이스를 통한 안전한 로컬 저장
