# Mobile App Review Checklist (심사 전용 운영 체크리스트)

앱 마켓(Play Store / App Store) 심사 제출 전에 운영팀, 개발팀, 디자인팀이 함께 확인하는 체크리스트입니다.

## 사용 가이드
- `현재 상태`: `미착수 / 진행중 / 완료 / 리스크` 중 하나로 표기
- `담당`: 실무 담당자 이름 또는 역할(예: Android Dev, iOS Dev, PM)
- `마감`: YYYY-MM-DD
- `증빙 파일`: 스크린샷, 설정 캡처, 릴리즈 노트, 정책 문서 등 저장 경로

## 운영 메모
- 제출 직전 24시간 내 체크리스트 상태를 최종 갱신합니다.
- `리스크` 상태 항목이 1개라도 있으면 제출 승인 전에 PM/릴리즈 오너 확인이 필요합니다.
- 증빙 파일은 공용 드라이브 또는 저장소 경로를 반드시 남겨 추적 가능하게 관리합니다.
