## Global Prompt Template
Use this template for consistent style across all cards:

```text
Rider-Waite inspired tarot card illustration, [CARD_NAME], full-body symbolic composition, ornate golden frame, mystical atmosphere, balanced lighting, high detail, clean line work, painterly texture, no text, no watermark, centered subject, vertical 2:3 ratio, style-consistent tarot deck
```

## File Naming Rule
- Output as PNG, 2:3 ratio (recommended 768x1152 or 1024x1536)
- Save by id:
  - `card_22.png` = Ace of Cups
  - ...
  - `card_77.png` = King of Wands
